Create a detailed lesson plan for a 90‑minute class on photosynthesis for high school students. The plan should include: (1) learning objectives, (2) materials needed, (3) hook/engagement activity, (4) direct instruction with key concepts, (5) guided practice (group work), (6) independent practice, (7) assessment (formative), and (8) closure. Include time estimates for each segment.
